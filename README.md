# 🎓 Student Management Mini Project

A simple Command Line Interface (CLI) application built with **Dart** to manage student records using Object-Oriented Programming (OOP) concepts.

This project was developed as a learning exercise to practice Dart fundamentals, OOP, abstraction, inheritance, interfaces, collections, and user interaction through the terminal.

---

## 🚀 Features

- Add a new student
- View all students
- Search student by name
- Update student information
- Delete a student
- Menu-driven CLI interface
- Simple and beginner-friendly code structure

---

## 🛠️ Built With

- Dart
- Command Line Interface (CLI)

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/sharifahmedd/student-management-mini-project.git
```

### 2. Open the project

```bash
cd student-management-mini-project
```

### 3. Run the program

```bash
dart run assignment.dart
```
